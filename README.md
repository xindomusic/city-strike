# City Strike 3D

A modern, high-tech 3D vertical scroller built with Node.js and plain JavaScript. Fly a helicopter over a procedurally generated city, avoiding obstacles and engaging targets.

## Features
- **3D Perspective City**: Skyscrapers with depth and height variations.
- **Node.js Engine**: Uses an Express server for reliable asset serving and local development.
- **Modern Aesthetic**: Neon wireframe graphics inspired by classic arcade shooters like 1943.
- **Fluid Controls**: Smooth movement and firing logic.

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```
3. Visit `http://localhost:3000` in your browser.

## Controls
- **Move**: WASD or Arrow Keys
- **Shoot**: Spacebar
